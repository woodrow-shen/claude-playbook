# /help-commands Command Guide

List all available slash commands with usage and descriptions.

## Usage

```
/help-commands
```

## What It Does

Displays a comprehensive list of all available commands with:
- Command name and usage syntax
- Brief description of what each command does
- Agents used by each command
- Best practices and workflow integration tips

## When to Use

- First time using claude-playbook in a project
- Quick reference for available commands
- Understanding which agents power which commands
