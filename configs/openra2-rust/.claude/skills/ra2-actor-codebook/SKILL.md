## Naming Conventions

- `ga*` = Allied structure (Allied prefix)
- `na*` = Soviet structure (Soviet prefix)
- `ca*` = Civilian structure
- `*.husk` = Destroyed/wreck variant
- `lo*` / `bridge*` = Bridge actors
- Generic names (power, refinery, barracks) = Base definition parents
